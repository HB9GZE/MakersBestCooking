# MakersBestCooking

A community cookbook — built by makers, for everyone.

## What is this?

**MakersBestCooking** is an open, collaborative cookbook where every maker is welcome to contribute their favorite recipes. Whether you are a weekend cook or a kitchen veteran, if you have a recipe worth sharing, this is the place for it.

## How to contribute

1. **Fork** this repository
2. **Copy** the recipe template from [recipe_template.md](recipe_template.md)
3. **Fill in** your recipe and save it as a new `.md` file (e.g. `pasta_carbonara.md`)
4. **Open a Pull Request** — others will review it and give feedback before it gets merged

## Recipe format

All recipes follow a common template to keep things consistent and easy to read. The template lives in [recipe_template.md](recipe_template.md).

## Guidelines

- One recipe per file
- Use the provided template so every recipe looks the same
- Add a category tag so recipes are easy to find (e.g. `vegan`, `meat`, `fish`, `no histamine`, `gluten-free`, ...)
- Constructive review comments are welcome — cooking is a craft, and we are all learning

## Everyone is welcome

No matter your skill level with cooking *or* with Git — jump in. This project is a great way to practice both.

Happy cooking!
