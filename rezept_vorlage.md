|---|---|
| **Kategorie** | vegan / Fleisch / Fisch / histaminarm / glutenfrei / ... |
| **Portionen** | 4 Personen |
| **Vorbereitungszeit** | 15 Min. |
| **Kochzeit** | 30 Min. |

---

## Zutaten & Zubereitung

| Zutaten | Zubereitung |
|:---|:---|
| | **Schritt 1 — kurze Bezeichnung** |
| 4 Eier | |
| 250 g Zucker | Eier mit Zucker schlagen, bis die Masse hell und cremig ist. |
| | |
| | **Schritt 2 — kurze Bezeichnung** |
| 200 g Mehl | |
| 1 TL Backpulver | Mehl und Backpulver sieben und vorsichtig unter die Eimasse heben. |
| | |
| | **Schritt 3 — kurze Bezeichnung** |
| Backofen auf 180 °C vorgeheizt | Teig in eine gefettete Form füllen und 30 Minuten backen. |
