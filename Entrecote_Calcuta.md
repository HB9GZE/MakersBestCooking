# ENTRECÔTE CALCUTTA

| | |
|---|---|
| **Kategorie** | Fleisch / asiatisch / Sweet and Sour |
| **Portionen** | 4 Personen |
| **Vorbereitungszeit** | 15 Min. |
| **Kochzeit** | 30 Min. |

---

## Zutaten & Zubereitung

| Zutaten | Zubereitung |
|:---|:---|
| | **Schritt 1 — Sauce** |
| 20 g Butter | In einer Saucenpfanne zergehen lassen.|
| 1 rote Peperoni | Peperoni in Streifen schneiden. |
| 3 Scheiben Ananas | Ananas in kleine Stücke schneiden. Zusammen mit den Peperonistreifen in der Butter dämpfen.|
| 1 kl. Büchse Eierschwämme (200 g) |  |
| 2 Esslöffel Mangochuttney | Eierschwämme zusammen mit Mangochuttney in die Sauce geben. Sofort Hitze reduzieren. |
| 0,2 l Tomatenketchup | In die Sauce geben. |
| 1 Teelöffel Currypulver | In die Sauce geben. |
| 1 Würfel Bouillon | In die Sauce geben. |
| Cayennepfeffer | Je nach belieben in die Sauce geben. |
| 3 Esslöffel Rahm | Sauce mit Rahm abschmecken. |
| | |
| | **Schritt 2 — Entrecôte** |
| Salz, Paprika, Öl, Pfeffer | |
| 4 Entrecôte (ca. 1,5 cm dick) | Entrecôte mit Öl, Salz, Paprika, Pfeffer einreiben. Anschliessend auf dem Grill oder Bratpfanne auf jeder Seite ca. 2 Minuten anbraten (Wenden sobald Saftperlen druchdringen). Anschliessend auf vorgewärmte Teller geben.|
| 3 Esslöfel Weisswein | Den Bratfond in Weisswein auflösen und der vorbereiteten Sauce beigeben |
| | |
| | **Schritt 3 — Anrichten** |
|Schnittlauch, Mandelsplitter | Die Entrecôte nach belieben mit Sauce übergiessen und mit Schnittlauch und Mandeslplitter bestreuen. |

---

## Anmerkungen

> Dazu passt Reis und erstaunlicherweise auch Rösti.
