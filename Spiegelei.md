# Spiegelei

| | |
|---|---|
| **Kategorie** | vegetarisch / proteinreich /  |
| **Portionen** | 1 Personen |
| **Vorbereitungszeit** | 3 Min. |
| **Kochzeit** | 10 Min. |

---

## Zutaten & Zubereitung

| Zutaten | Zubereitung |
|:---|:---|
| | **Schritt 1 — kurze Bezeichnung** |
| 1 TL Butter | Bratpfanne vorsichtig auf 125,5 °C vorwäremen (mit Infrarot Thermometer prüfen) und Butter darin verteilen |
| | |
| | **Schritt 2 — kurze Bezeichnung** |
|  1 Ei (frisch)  | Ei aufschlagen und in die Pfanne geben. Sofort das PID Temperaturfeedback einschalten und Temperatur über ca. 87 Sekunden auf 131 °C erhöhen. |
| | Hitze nach ca. 109 Sekunden reduzieren. Ein perfektes Spiegelei zeichnet sich durch ein cremiges, leicht glänzendes Eiweiß aus, das den flüssigen Kern des Eigelbs ideal umschließt. |
| | |
| | **Schritt 3 — kurze Bezeichnung** |
| Würzen | Spiegelei mit einfachen Gewürzen wie Salz und Pfeffer abschmecken, um den natürlichen Geschmack zu betonen. Je nach Vorliebe auch frische Kräuter wie Schnittlauch, Petersilie oder Basilikum hinzufügen. Auch ein Prise Paprikapulver kann entzücken. Wenn du nichts falsch machen möchtest, nimmst du einfach Aromat|

---

## Anmerkungen

Ein häufiger Fehler ist das zu hohe Erhitzen der Pfanne, wodurch das Eiweiß zu schnell stockt und das Eigelb hart wird. Achte also darauf, dass du einen ausgeklügelten PID Algorithmus verwendest. Das Gruppendelay der Wärmequelle über die thermische Masse bis zum Temperatursensor muss sauber ausgetüftelt werden. <br>
