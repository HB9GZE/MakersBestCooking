# Getränkter Zitronencake
Author:  Stefan
| | |
|---|---|
| **Kategorie** | vegan |
| **Portionen** | 4 Personen |
| **Vorbereitungszeit** | 30 Minuten |
| **Kochzeit** | 60 Minuten bei 180°C |

---

## Zutaten und Zubereitung

| Zutaten | Zubereitung |
|:---|:---|
| | **Schritt 1 — Zucker und Ei** |
| 250g Butter weich |Ofen auf 180°C vorheizen. Butter in eine Schüssel geben, Zucker und Salz darunter rühren. |
| 4 Eier |Ein Ei nach dem anderen darunterrühren, weiterrühren bis die Masse heller ist. |
| 250 g Zucker |Von 2 Zitronen die Schale dazureibe, darunterrühren. |
| 1 Prise Salz | |
| 3 Bio Zitronen | |
| | |
| | **Schritt 2 — Mehl und Backpulver** |
| 250g Mehl |Mehl und Backpulver mischen und unter die Masse rühren.  |
| 1 TL Backpulver|Teig in die vorbereitete Form füllen. |
| | |
| | **Schritt 3 — Backen** |
|  |Backen: ca. 1 Stunde in der unteren Hälfte des Ofens. Herausnehmen |
|  |Cake in der Form etwas abkülne lassen. Mit einem Holzspiesschen dicht einstechen. |
| | **Schritt 4 — Finish** |
| 100g Puderzucker |Alle Zitronen auspressen (ergibt ca. 1 dl Saft), Puderzucker darunterrühren,|
||über den Cake giessen. Aufsaugen lassen. Cake aus der Form nehmen.|

---

## Anmerkungen

> Verwende eine 30cm Cake-Form. Mit Backpapier auskleiden.
> Haltbarkeit: im Kühlschrank zugedeckt 3 Tage. Der Kuchen vor dem servieren auf Zimmertemperatur aufwärmen lassen.
